CSS BOX MODEL

A CSS Box Model is a wrap/cover around every HTML element.
It is used to create the design and layout of webpages.
It can be used also as a toolkit for customizing the layout of different elements
PROPERTIES OF CSS BOX MODEL

  1. Content: it is the content of the box where images,videos,audios and text appear
  2. Padding: it is a space surrounding content.And it is transparent
  3. Border: it is the line surrounding content and padding and separating them from the margin
  4. Margin: it clears an area outside the border. And it is transparent
  5. Width: it specifies the width of element's content area
  6. Height:it specifies the height of element's content area
